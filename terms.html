TEKNOMEDYAM.COM Sitesine üye olan herkes sözleşmeyi kabul etmiş sayılırlar.


1) Ödeme İadeleri : Yüklenilen bakiye Teknomedyam.com içerisinde harcanabilir. İade yapılmamaktadır.

2) 1 - a)  Yanlış Sipariş İadeleri :  Yanlış siparişlerin iadeleri yoktur. Yanlış girilen link bulunamadığında kullanıcıya bakiyesi iade edilir eğer girilen link veya kullanıcı adına gönderim sağlandıysa iade yapılmayacaktır. Sipariş girerken tüm sorumluluk kullanıcıya aittir.

3) 2)  Sipariş Tamamlanma Süresi : TEKNOMEDYAM.COM sitesi manuel işlemlerinin yanında diğer sitelerden api ile servis çekip kullanıcılarına sunmaktadır. TEKNOMEDYAM.COM sitesinin aracılık yaptığı servislerin gönderim sürelerinde geçikme gibi durumlar olabilir. Kullanıcılar sipariş verirken bunları göz önünde bulundurmalıdır.  

4) 2 - a ) Api ile çekilen servisler : Yabancı takipçi, izlenme, beğeni gibi servislerin hepsi api ile çekilen servislerdir. Bu servislere müdahale hakkımız yoktur. Girilen sipariş iade - iptal edilemez.

5) 3 ) Yapılan ödemeler : Sitemiz paywant alt yapısı ile ödeme almaktadır. Yaptığınız ödemeler otomatik olarak hesabınıza tanımlanır. Paywanttan kaynaklı olarak hesabınıza yansıyan fazla bakiye tespit edildiği takdirde uyarı dahi yapılmadan silinecektir. Eksik yüklenen bakiyeleri destek talebinden bildirirseniz TEKNOMEDYAM.COM sitesi gerekli incelemelerden sonra bakiyenizi eklemeyi taahhüt eder.  

6) 4 ) TEKNOMEDYA.COM sitesi manuel işlemlerini sadece bot (gerçek olmayan) hesaplar üzerinde kullanmaktadır. Gerçek kişilerin hesaplarını ele geçirmez ve gerçek hesaplar üzerinden takip, beğeni işlemi yapmaz. Api ile çekilen servislerde ki takipçi, beğeni işlemlerinde kullanılan hesaplar (gerçek hesap ise) tüm sorumluluk apiyi sağlayan sitededir. TEKNOMEDYAM.COM sitesi sadece aracılık yapmaktadır.

7) 5) TEKNOMEDYAM.COM sitesi istediği kullanıcı ile çalışma hakkını gizli tutmaktadır.  TEKNOMEDYAM.COM sitesi üslubuna dikkat eden, saygılı müşteriler ile çalışmaya özen gösterir ve bu kurala uymayan kullanıcıların bakiyesini iade edip üyeliğini silebilir.

8) 6) Kayıt olan kullanıcılar TEKNOMEDYAM başlıklı bilgilendirme smsleri almayı kabul etmiş sayılır.

9) 6-b) Kayıtlı kullanıcının isteği üzerine TEKNOMEDYAM.COM sitesi sms göndermeyeceğini taahhüt eder.

10) 7) Kayıt olan bütün kullanıcılar sözleşmeyi kabul etmiş sayılır.
